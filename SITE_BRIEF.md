# QIRA Peptide Site — Build Brief

## Goal
Build a premium peptide/optimization landing site concept with physician-led authority, luxury performance aesthetics, concise declarative copy, and custom peptide-inspired animation language.

## Strategic position
Per DeerFlow research, the brand should sit between commodity telehealth GLP-1 sellers and bro-science peptide shops. The brand should feel like Park Avenue endocrinology translated into a high-performance luxury digital experience.

## Tone
- Short declarative lines
- Physician-led authority
- Luxury performance
- Transformation without hype
- Sophisticated, not biohacker-cheesy
- Outcome-oriented, not commodity-drug oriented

## Core copy direction
- Codes, not compounds
- Stacks as delivery bundles
- Labs and physician oversight as trust anchors
- Dr. Clifton Jackness / Park Avenue Endocrinology as the medical authority block
- Tagline direction: Look better. Feel stronger. Stay online.
- Related transformation language can be explored, but copy should remain original and premium

## Inspirations
- DeerFlow peptide market/positioning research from `../deerflow-peptide-research-report.md`
- `https://systemlabs.com/` for premium motion/hero animation ambition
- `https://apollohousenyc.com/services/peptide-therapy/` for physician-led peptide framing
- User also referenced MindJoin, Magic Lantern, and Portion websites, but exact URLs are still unresolved and should be treated as pending references until confirmed

## Visual direction
- Editorial luxury + medical precision
- Dark or near-dark base palette with crisp ivory, silver, muted gold, and subtle biotech glow accents
- Motion should evoke peptides / signaling / molecular choreography, not generic floating blobs
- Create a top-of-page hero animation placeholder inspired by the presence of the System Labs motion language
- Create custom in-page animation concepts that feel peptide-like: chain assembly, signal pulses, molecular arcs, responsive strands, restrained particle behavior

## Sections to build
1. Hero with strong headline, subhead, CTA, and animation placeholder
2. Codes overview grid/carousel
3. Physician authority / CMO section
4. Why this is different / protocol-led care section
5. How it works: Labs -> Consult -> Code -> Consistency
6. Featured Codes / Stacks
7. Compliance-aware footer / medical disclaimers placeholder

## Codes to include initially
- Glow Code
- Weight Code
- Sustainability Code
- Recovery Code
- Performance Code
- Longevity Code
- Focus Code
- Hormone Code
- Sleep Code

Each code should generally use:
- name

## Compliance guardrails
- Avoid disease claims
- Use support / optimization / resilience language
- Be cautious with BPC-157 / TB-500 presentation
- Treat compounded GLP-1 language carefully
- Include physician approval / clinical necessity framing
